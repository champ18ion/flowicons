# FlowIcons Playground 🎡

This is the interactive laboratory for **FlowIcons**. Here you can experiment with animations, physics, and styles before integrating them into your project.

## 🛠️ Features
- **Real-time Customization**: Adjust size, stroke, and color on the fly.
- **Animation Toggles**: Trigger loading states to see the "Coca-Cola Standard" in action.
- **Code Generation**: Click any icon to copy the exact JSX code with your current settings.

## 🚀 Development
The playground is linked to the library source via **npm workspaces**. 

To run locally:
```bash
npm run dev -w playground
```

## 🏗️ Architecture
Built with **Vite**, **React 19**, **Tailwind CSS**, and **Framer Motion**.
